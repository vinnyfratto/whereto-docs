# Week in Review

One file per ISO week (`YYYY-Www.md`), written by the weekly agent (see
[../docs/automation/weekly-review-agent.md](../docs/automation/weekly-review-agent.md)).
Each entry covers development, bug fixes, and (always) regressions and debt.

These accrete into the historical record a reviewer reads to gauge velocity, discipline, and
how failures are handled. They also feed [../CHANGELOG.md](../CHANGELOG.md).

To generate this week's entry: run `/weekly-review` in Claude Code, or
`node scripts/weekly-report-data.mjs` and follow the agent spec.

The first real entry will be created on the next Friday run.
